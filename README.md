# 研路通 - 研究生日常管理与交流系统 (GradManager)

这是一个专为实验室和课题组设计的**研究生日常管理与学术交流系统**。系统基于现代化的前端设计，采用深色玻璃拟物化风格（Glassmorphism），具备出色的响应式布局与流畅的交互动效，致力于为导师和研究生提供一站式的日常协同空间。

## 🌟 核心功能板块

1. **工作仪表盘 (Dashboard)**
   - **每日签到率统计**：动态计算今日考勤进度条。
   - **双板块速览**：快速浏览最新公告和论坛热议帖子。
   - **签到趋势折线图**：通过精美的动态 SVG 曲线渲染过去一周内实验室的到岗趋势。
   
2. **考勤打卡终端 (Daily Sign-in)**
   - **巨型拟真指纹打卡按钮**：支持一键签到，伴有水波纹（Ripple）反馈动效。
   - **打卡参数配置**：可自主选择签到区域（402实验室、405会议室、计算中心等）并填写实验或学习备注。
   - **智能时段划分**：系统自动根据时间戳识别上午（AM）打卡或下午（PM）打卡。
   - **打卡榜单**：实时展示今日已签到成员、打卡时间及备注。
   - **历史考勤日历**：月度视图排版，绿色标记“已签到”、红色标记“工作日缺勤”、外框高亮“今日”。

3. **通知公告栏 (Announcements)**
   - **分类浏览**：支持按“重要”、“学术”、“活动”、“系统”筛选公告。
   - **动态发布公告**：管理员角色可以发布系统通告，发布后将自动推送到所有学生的未读消息队列。
   - **未读气泡反馈**：卡片边缘带有金色呼吸条，醒目标示未读内容，点击阅读后自动消退。

4. **学术科研论坛 (Academic Forum)**
   - **主题发帖**：支持设定帖子标题、板块分类（学术科研、日常生活、项目进展）以及自定义标签（如 `#PyTorch`、`#CVPR`）。
   - **详情与回帖**：点击帖子可进入详情弹窗，查看完整长文并参与实时讨论。
   - **话题云 (Tag Cloud)**：侧边栏展示当前热门科研学术话题，增强实验室学术氛围。

5. **成员名册 (Member Directory)**
   - **详细信息卡片**：展示导师及所有研究生的姓名、年级、学号、主要研究方向和邮箱。
   - **搜索筛选**：支持按照姓名、身份、研究方向进行实时模糊检索。
   - **新增成员**：提供可视化表单快速录入新入校的研究生。

## 🛠️ 技术实现与架构

- **核心语言**：HTML5 语义化结构 + Vanilla JavaScript 业务逻辑。
- **视觉系统**：纯 Vanilla CSS3 自定义变量系统，结合 CSS Grid 与 Flexbox 弹性布局，配备了玻璃拟物滤镜 (`backdrop-filter`)、光影过渡 (`box-shadow`) 以及多层次漂浮景深 Blob 泡泡背景。
- **数据持久化**：所有打卡记录、新发的帖子、新的回帖、添加的成员和公告均实时同步至浏览器的 `localStorage` 中。即使刷新页面，您添加和修改的数据也将被完整保留。
- **运行环境**：无任何第三方构建依赖，开箱即用。

## 🚀 如何运行

### 方式一：一键启动（推荐）

直接双击 **`start.bat`** 即可。脚本将自动完成：
1. 检查 Python 环境
2. 安装所有依赖（`flask`, `waitress`, `flask-limiter` 等）
3. 生成安全的 `SECRET_KEY`
4. 以生产模式启动服务器并显示局域网访问地址

---

### 方式二：手动启动（精确控制）

**Step 1：安装依赖**
```bash
pip install -r requirements.txt
```

**Step 2：设置安全密钥（必须！）**

Windows PowerShell：
```powershell
$env:SECRET_KEY = (python -c "import secrets; print(secrets.token_hex(32))")
```

Linux / Mac：
```bash
export SECRET_KEY=$(python -c "import secrets; print(secrets.token_hex(32))")
```

**Step 3：启动服务器**
```bash
python server.py
```

启动成功后，终端将显示本机和局域网访问地址，例如：
```
=======================================================
🚀  研路通 研究生管理系统 已启动（生产模式）
   本机访问：http://localhost:8181
   局域网访问：http://192.168.1.x:8181
   按 Ctrl+C 停止服务
=======================================================
```

---

### 方式三：开发模式（调试用，不建议生产）

```bash
$env:FLASK_DEBUG = "true"   # Windows PowerShell
python server.py
```

## 🔐 安全说明

- 系统已内置 **Session 加密**（通过 `SECRET_KEY` 保护用户登录状态）
- **登录接口限速**：每 IP 每分钟最多 10 次，每小时最多 50 次，防暴力破解
- **密码修改功能**：点击侧边栏左下角的 🔑 按钮修改密码，修改后强制重新登录
- 所有用户密码均以 **bcrypt 哈希**存储，不保存明文密码
- SQLite 已开启 **WAL 模式**，支持多用户并发安全写入
